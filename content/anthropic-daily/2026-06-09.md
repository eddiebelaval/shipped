### D. Claude Apps

#### 2026-06-09 -- Claude for Apple Foundation Models framework (announced) ([claude.com/blog/claude-for-foundation-models](https://claude.com/blog/claude-for-foundation-models))
`[APPS]`

A Swift package that makes Claude available as a server-side language model in Apple's Foundation Models framework. Conforms to the `LanguageModel` protocol so developers can drive it with `LanguageModelSession` alongside Apple's on-device model. Supports multi-step reasoning, code generation, web search, code execution, and SwiftUI streaming. Targets iOS 27, iPadOS 27, macOS 27, visionOS 27, watchOS 27. Announced June 9, available June 10.

**How to use:** Add the Swift package; see [API docs](https://platform.claude.com/docs/en/cli-sdks-libraries/libraries/apple-foundation-models).
