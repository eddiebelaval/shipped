#### 2026-06-25 -- Claude Code v2.1.193 ([releases](https://github.com/anthropics/claude-code/releases))
`[CODE]`

Auto-mode gets a new `autoMode.classifyAllShell` setting that routes all Bash and PowerShell commands through the classifier, not just arbitrary-code-execution patterns. A new `claude_code.assistant_response` OpenTelemetry log event emits model response text as a structured log, redacted by default, opt-in via `OTEL_LOG_ASSISTANT_RESPONSES=1`. **How to use:** `claude update` or reinstall. Set `autoMode.classifyAllShell: true` in your Claude settings to route all shell commands through the classifier. Set `OTEL_LOG_ASSISTANT_RESPONSES=1` to enable response logging for your observability pipeline.
