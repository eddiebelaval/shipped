## Release Log

### APPS

#### 2026-06-17 -- Claude Design: design system imports, code sync, canvas editing ([announcement](https://claude.com/blog/claude-design-stays-on-brand-for-daily-work))
`[APPS]`

Claude Design now imports design systems from GitHub repositories, design files, or raw uploads. Bidirectional sync with Claude Code via `/design-sync` and `/design` commands covers the design-to-code handoff. Direct canvas editing and export connectors to Adobe, Canva, and Miro ship alongside an admin role that locks company components. Available to Claude Pro, Max, Team, and Enterprise customers.

**How to use:** Open Claude Design from your Claude account. Use `/design-sync` to push changes to Claude Code; use `/design` to start from a running codebase.
