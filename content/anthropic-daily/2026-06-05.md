## Lead

**Claude Code v2.1.166: fallback models and a quiet security tightening**

The feature is `fallbackModel`. Configure up to three alternative models tried in order when your primary model is overloaded. Production agent teams have been building this manually for months. Now it is a config setting.

The same build hardens cross-session messaging. Relayed messages no longer carry user authority. Quiet, non-headline, load-bearing. Glob patterns now work in deny rule positions, so a single `"*"` blocks every tool in one line. Retry on unexpected non-retryable API errors is in. Fixes cover JetBrains IDE flickering (2026.1+), remote session registration hangs, and Kitty keyboard protocol Shift+non-ASCII drops.

v2.1.165 shipped earlier the same day. Bug fixes only. Nothing user-facing.
