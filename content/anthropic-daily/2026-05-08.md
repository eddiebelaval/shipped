#### 2026-05-08 — [Claude Code v2.1.136](https://github.com/anthropics/claude-code/releases)
`[CODE]`

Roughly 50 fixes in one release. New capability: `settings.autoMode.hard_deny` lets teams define auto-mode classifier rules that block unconditionally, regardless of user intent or configured allow-exceptions. New enterprise telemetry hook: `CLAUDE_CODE_ENABLE_FEEDBACK_SURVEY_FOR_OTEL` re-enables the session quality survey for orgs capturing responses via OpenTelemetry. WSL2 image paste from Windows clipboard now works via PowerShell fallback.

Notable fix cluster: MCP servers configured in `.mcp.json`, plugins, and claude.ai connectors were silently disappearing after `/clear`. OAuth token race that could cause a rare login loop. `--resume` and `--continue` not finding sessions when the project path contains underscores.

**How to use:** `claude update` or reinstall. Add `settings.autoMode.hard_deny` rules in `.claude/settings.json` to define unconditional classifier blocks for your team.
