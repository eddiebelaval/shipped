### Claude Code

#### 2026-06-29, Claude Code v2.1.196 ([github.com/anthropics/claude-code/releases/tag/v2.1.196](https://github.com/anthropics/claude-code/releases/tag/v2.1.196))
`CODE`

Broad reliability and quality drop. Org admins can now set default models in the console, surfaced as "Org default" or "Role default" in the model picker. Sessions get readable names on start. File attachments are clickable in chat (Cmd/Ctrl-click opens in Finder or Explorer). Streaming idle watchdog on by default, aborting and retrying if a stream produces no events for five minutes. Per-frame render work cut by roughly 37%.

Duplicate recap lines after background sessions. MCP OAuth scope requests on GitLab self-hosted and other enterprise IdPs now request only required scopes. PowerShell `git diff`/`git grep` and quoted patterns with `|` no longer false-positive on exit 1. Keyboard focus stuck when opening the claude agents panel. Mid-turn crash recovery for Remote sessions. `/context` showing 0 tokens on Bedrock. Set `CLAUDE_ENABLE_STREAM_WATCHDOG=0` to disable the new watchdog.
