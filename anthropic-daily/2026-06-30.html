Lead

Sonnet 5 is the new default

Claude Sonnet 5 launched June 30 and immediately became the default model for every Claude Free and Pro user. Not a preview. Not a limited rollout. Default, as of today.

The positioning is explicit: Sonnet 5 is an agentic model. It can plan, use tools like browsers and terminals, and run autonomously at a level that until recently required Opus-class compute and Opus-class pricing. On agentic benchmarks it runs close to Opus 4.8. The context window is one million tokens, native.

Pricing is the other story. Through August 31, 2026: $2 per million input tokens, $10 per million output tokens. After that, standard pricing at $3/$15. That introductory window is real. If you are building long-running agents and have not recalculated, now is the time.

Available today: Claude API (model ID claude-sonnet-5), claude.ai (new default for Free and Pro), Claude Code (update to v2.1.197), Amazon Bedrock, Google Cloud, and Microsoft Foundry. The system card published the same day.

Sonnet 5 is now the default model. New in this release: org-configured default models, readable session names, clickable file attachments, and a smoother agents view. 2026-06-29
CODE
Claude Apps Gateway for Amazon Bedrock and Google Cloud
A self-hosted control plane for Claude Code deployments on enterprise cloud infrastructure. Runs as a single stateless container on Linux, backed by PostgreSQL. Gives organizations: corporate SSO login, centrally enforced policy, role-based access, per-user cost attribution, and daily/weekly/monthly spend limits per org, group, or user. The gateway holds the upstream credential and routes inference to the Claude API, Bedrock, or Google Cloud, with optional failover between providers. Does not forward inference traffic or usage data to Anthropic unless configured to use the Claude API directly.
How to useSee the Claude apps gateway docs. Deploy as a container, configure your upstream provider credential, set policies and spend limits.
Why it mattersWith the gateway covering Bedrock and Google Cloud, and Claude in Microsoft Foundry going GA the same week, Anthropic now has enterprise-grade deployment across all three major hyperscalers with a self-hosted control plane for each.
Claude Apps

APPS
Claude Science (beta)
A desktop application for scientific research, now in beta on macOS and Linux for Pro, Max, Team, and Enterprise subscribers. Over 60 curated connectors and skills pre-configured for genomics, single-cell analysis, proteomics, structural biology, and cheminformatics. Claude writes and runs Python, R, or shell code in a sandboxed environment, pulls from scientific databases, and saves results as versioned artifacts with full provenance. Every output carries a complete audit trail. Anthropic is funding up to 50 research projects at up to $30,000 in credits each; applications open through July 15, 2026.
How to useAvailable at claude.com/product/claude-science for Pro, Max, Team, and Enterprise plans. Discounted seats for academic and nonprofit labs.
Why it mattersAnthropic ran The Briefing: AI for Science event in San Francisco the same morning (livestreamed, 9 AM PST). Life sciences is Anthropic's declared second vertical after developer tooling. The product mirrors Claude Code: domain-specific, autonomy-forward, designed for long-horizon tasks from concise instructions.
