John Jumper Joins Anthropic

John Jumper, 2024 Nobel Prize in Chemistry laureate and co-creator of AlphaFold, announced June 19 that he is leaving Google DeepMind after nearly nine years to join Anthropic. Bloomberg and CNBC reported the move on June 19; TechCrunch covered it on June 20. No title or start date has been announced.

AlphaFold solved protein structure prediction. Jumper is the scientist most identified with that achievement. He is now at Anthropic.

Claude Code v2.1.183
Auto-mode safety tightened: destructive git commands (git reset --hard, git checkout -- ., git clean -fd, git stash drop) are now blocked unless the request explicitly asked to discard local work. git commit --amend is blocked when the commit was not made by the current agent session. terraform destroy, pulumi destroy, and cdk destroy are blocked unless that specific stack was named in the request. Adds a deprecation warning when the requested model is being automatically upgraded. Adds attribution.sessionUrl setting to omit the claude.ai session link from commit and PR metadata. Adds /config --help. Bug fixes cover WebSearch in subagents, terminal cursor behavior, fullscreen TUI corruption, and MCP server auth exposure.
How to useclaude update or reinstall from npm. New auto-mode safety rules are on by default.
