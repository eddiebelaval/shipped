Claude Code ships Agent View and a goal-based continuation command; four language SDKs sync around AWS client support and API-key log hygiene.

Lead

Claude Code v2.1.139 landed at 18:43 UTC. Two features anchor it. Agent View (Research Preview) surfaces every session, running, blocked, or completed, in a single list accessible via claude agents. The /goal command lets you set a completion condition once; Claude keeps working across turns until it hits it, with a live overlay showing elapsed time, turns, and tokens. Forty-plus bug fixes ride along, including a deadlock fix for expired credentials and a hard 16 MB cap on previously unbounded memory growth in HTTP/SSE MCP servers. (release notes)
