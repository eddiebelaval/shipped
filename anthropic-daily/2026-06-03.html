Introducing the Services Track and Partner Hub of the Claude Partner Network

Three months after launching the Claude Partner Network, Anthropic formalized it with tiers and a portal. The Services Track has three levels:

  • Select: 10 certified practitioners, 2 production deployments, 1 public endorsement
  • Preferred: 100 certified practitioners, 15 active customers, 3 endorsements
  • Global Premier: 1,000 certified practitioners, 100 customers across 3+ regions, 15 endorsements, joint executive-backed business plan

The Claude Partner Hub gives partners a daily-refreshed view of where they stand against requirements. Customers use it to find qualified firms for their scope. Since the March 2026 launch: 40,000+ firms applied, 10,000+ consultants certified.

Claude Code
2026-06-03 CODE
Claude Code v2.1.162

New features: claude agents --json includes a waitingFor field showing what a waiting session is blocked on (e.g., a permission prompt); --tools with explicit Grep/Glob now provides dedicated native embedded search; /effort confirms when a chosen level will persist as default for new sessions; slash commands in the autocomplete menu now fill into the prompt rather than running immediately; Remote Control shows as a persistent footer pill with session link instead of a startup message. Windsurf renamed to Devin Desktop across the IDE menu, /terminal-setup, and /scroll-speed.

Bug fixes: silent startup hang when the config directory is read-only or unwritable; WebFetch permission rules not applying to preapproved domains; Windows permission rules with backslashes or case-variant paths; Esc interrupt at turn start being silently dropped in stream-json and SDK sessions; API 400 errors for classifier queries and MCP server descriptions with emoji near truncation points; MCP timeout config values below 1000ms being floored to 1 second; LSP workspaceSymbol returning no results; claude agents status and session name truncation on wide terminals; Ctrl+V image paste not working in dispatch and reply boxes; background sessions losing conversation when the background service fails; cross-session messaging breaking with deep CLAUDE_CODE_TMPDIR paths; opening a running background session stalling for 5 seconds.
