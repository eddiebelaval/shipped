Lead

Claude Code v2.1.147: 28 fixes and a renamed command

The most visible change in v2.1.147 is a rename: /simplify is now /code-review, and it ships with effort levels (/code-review high) and a --comment flag that posts findings directly as inline PR comments on GitHub. Clean integration, no clipboard archaeology.

The second change worth tracking is operational: pinned background sessions (Ctrl+T in claude agents) now survive idle, restart in place when an update lands, and are shed under memory pressure only after non-pinned sessions clear first. Sessions that hold their state through an update cycle are sessions you can rely on.

The rest is a 28-item fix list covering PowerShell on Windows, MCP server pagination, auto-updater retries, diff rendering performance, and edge cases in plugin and shell snapshot handling. Nothing changes what the tool can do. Several things change how reliably it does it.

2026-05-21 Claude Code v2.1.147
CODE

/simplify renamed to /code-review with effort-level reporting (/code-review high) and a --comment flag for GitHub PR inline comments. Pinned background sessions (Ctrl+T in claude agents) now survive idle and restart in place when updates land. 28 additional fixes including: auto-updater retry logic, diff rendering performance on large files, enterprise login enforcement for forceLoginOrgUUID and forceLoginMethod, PowerShell on Windows (winget and Microsoft Store installs), MCP server pagination (resources and prompts past page 1 were silently dropped), shell snapshot user-function handling, plugin agent multi-type declarations, hook if condition matching, and full-screen strobing on Windows Terminal.

How to use: Run claude update to get v2.1.147. Then use /code-review in any session, or /code-review high for a deeper pass. Add the --comment flag to post findings as inline GitHub PR review comments.

SDK-PY

Adds thinking-token-count beta support: when streaming extended thinking, the SDK now surfaces estimated token counts in thinking block deltas. Useful for cost estimation and context budget management in agentic loops that rely on extended thinking.

How to use: Include the thinking-token-count beta header in your request. Token estimates arrive as metadata on thinking block delta events. See streaming docs for the full schema.
