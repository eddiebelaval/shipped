Claude Code v2.1.181 ships a new in-prompt config command and a 30-fix maintenance pass.

The day's confirmed release is Claude Code v2.1.181, which ships /config key=value as a new in-session command. Set thinking=false, switch effort levels, or change the model without leaving the prompt. Works in interactive, -p, and Remote Control modes.

Two startup issues are patched. A regression from 2.1.169 was adding roughly 120ms per launch in fresh environments, silently eating time on every session open. A second blocker could freeze the terminal for up to 15 seconds when the account settings fetch stalled on a degraded network.

Prompt caching on Foundry and custom ANTHROPIC_BASE_URL endpoints is also fixed, a quiet but operationally important patch for anyone routing Claude through a custom gateway. Foreground subagents now respect the same five-level nesting cap as background subagents, closing an unbounded-chain correctness issue that could surface in complex agentic workflows.

CODE
Claude Code v2.1.181
New: /config key=value sets any setting from the prompt in interactive, -p, and Remote Control modes. sandbox.allowAppleEvents opt-in for macOS Apple Events. CLAUDE_CLIENT_PRESENCE_FILE env var suppresses mobile push notifications. Bun runtime upgraded to 1.4. Streaming of long paragraphs is now line-by-line. API connection drops mid-thinking now auto-retry. Subagent panel auto-hides idle agents after 30 seconds, caps at five rows with scroll hints. MCP OAuth browser page matches Claude Code visual style and auto-closes on success.
Fixed: prompt caching on custom ANTHROPIC_BASE_URL and Foundry. Write/Edit producing zero-byte or truncated files on network drives and cloud-synced folders. Apple Events entitlement issues on macOS. Startup regression from 2.1.169 (~120ms per launch in fresh environments). Startup blocking for up to 15 seconds on degraded networks. Startup crash when .claude.json contains corrupted null entries. macOS TUI freezing when Spotlight is reindexing. Idle session history lost when the 30-day cleanup process runs. Foreground subagents now capped at five nesting levels. /recap and conversation forks using the previous model after a switch. AWS credential refresh loops. /stats date display off by one day in UTC-negative timezones. Thirty-plus additional UI, clipboard, and window management fixes.
How to use Run claude update or reinstall. The /config command is immediately available. Example: /config thinking=false
