Anthropic held its first developer conference today, shipped a Code Review feature that every internal team already runs, and signed SpaceX for 300 megawatts of compute at the Colossus 1 data center in Memphis.

Lead

The "Code with Claude 2026" keynote opened in San Francisco at 09:00 PT with CPO Ami Vora saying the event "is about how we are making our products work better for you." No new model. What shipped instead: Code Review, a tool that dispatches a team of parallel agents against a pull request, each checking for different error categories, leaving comments with conclusions and fix suggestions. Anthropic said it is already used by every team at the company.

The same morning, Anthropic published a separate announcement: it has signed SpaceX for all AI capacity at Colossus 1 in Memphis, giving Anthropic access to over 220,000 NVIDIA GPUs drawing more than 300 megawatts. The deal doubles Claude Code rate limits for paid plans, removes peak-hour caps for Pro and Max subscribers, and sharply increases Opus API throughput. Bloomberg first reported the deal.
