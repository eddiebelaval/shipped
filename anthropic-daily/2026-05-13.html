Anthropic shipped Claude for Small Business, two Claude Code builds, and the same cache-diagnostics-plus-ManagedAgents drop across both SDK flavors.

Lead

Claude for Small Business is a toggle install. That is the product decision underneath the announcement: no new interface, no separate app, no extra line item. The package drops Claude-powered connectors and agentic workflows into the software small businesses already run: Intuit QuickBooks, PayPal, HubSpot, Canva, DocuSign, Google Workspace, Microsoft 365. Payroll planning, month-end close, cash-flow forecasting, invoice chasing, and campaign creation are the covered workloads. (anthropic.com/news/claude-for-small-business; reported by TechCrunch and The Register, May 13.)

Anthropic put a number on the audience: 44% of U.S. GDP, roughly half the private-sector workforce. The gap they are pitching against is the distance between "we have a chat window" and "Claude is doing work." Starting May 14 in Chicago, a 10-city free half-day workshop tour runs to close that gap in person. No extra cost beyond existing Claude licenses. Anthropic is also backing a Workday Foundation solopreneurship accelerator and partnering with three CDFIs (Accion Opportunity Fund, Community Reinvestment Fund USA, Pacific Community Ventures) with Claude credits.

The connector model is the tell: Anthropic is not building the SMB integrations itself. It is providing the reasoning layer and the workflow primitives; the software partners carry the integration surface. Same structure as the enterprise push.

Claude Code v2.1.141
Agent view (Research Preview) shows all Claude Code sessions in a single list: running, blocked on you, or done. /goal sets a completion condition and Claude keeps working across turns until it is met. /scroll-speed tunes mouse wheel speed with live preview. claude plugin details <name> shows a plugin's component inventory and projected per-session token cost. Background agents via /bg or double-left-arrow preserve the current permission mode. Spinner feedback goes amber after 10 seconds of long thinking. Plus reliability fixes across background side-queries, daemon status, agent dashboard, permission prompts, markdown tables, and editor integrations.
How to use: claude update or reinstall. Agent view appears automatically for Research Preview users. /goal <condition> starts a persistent multi-turn run.

Agent tool subagent_type matching now accepts case- and separator-insensitive values. Updated agent color palette.
