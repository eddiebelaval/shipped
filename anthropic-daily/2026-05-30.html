Claude Code v2.1.158

Auto mode is now available on Bedrock, Vertex, and Foundry for Opus 4.7 and Opus 4.8. Opt in by setting CLAUDE_CODE_ENABLE_AUTO_MODE=1. Previously, Auto mode was limited to direct API subscribers and Claude.ai plan holders. Enterprise cloud customers running Claude through AWS Bedrock, Google Cloud Vertex AI, or Anthropic Foundry can now enable it without migrating off their existing platform.
