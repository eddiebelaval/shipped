Mythos went public today, renamed Fable 5, with per-session safety routing that fires in less than one in twenty sessions.

The Public Frontier

Fable 5 posts 80.3% on SWE-Bench Pro, eleven points clear of the field, and spatial reasoning that nearly tripled over Opus 4.8. Stripe estimated two-plus months of engineering time on a 50-million-line Ruby migration; Fable 5 finished in a day. The underlying model is Mythos 5, filtered through safeguards that route roughly one in twenty sessions to Opus 4.8 when queries touch high-risk territory (cybersecurity, biology). Model ID: claude-fable-5. Pricing: $10/$50 per million tokens. Free on paid plans through June 22, then usage credits until capacity catches up.

Mythos 5 is the same model with some safeguards lifted, deployed through Project Glasswing for authorized cyberdefenders and infrastructure providers. The public gets Fable. The cyberdefense community gets Mythos.

Claude Fable 5 and Claude Mythos 5
Anthropic's strongest publicly available model, built on Mythos-class intelligence with safeguards active. Safeguards fire in less than 5% of sessions; affected queries fall back to Opus 4.8. Mythos 5 is the same model with some safeguards lifted, deployed through Project Glasswing for authorized cyberdefenders and infrastructure providers.

SWE-Bench Pro: 80.3% (next model: 69.4%). FrontierCode Diamond: 29.3% vs 13.4% for Opus 4.8. Spatial reasoning: 38.6% vs 14.5%. Pricing: $10/$50 per MTok input/output. Free on Pro, Max, Team, and Enterprise through June 22, then usage credits.
How to use model="claude-fable-5" on the Claude API. Available on Amazon Bedrock; Vertex AI and Foundry to follow.

Jun 08
Code
Claude Code v2.1.169
Safe mode, directory change, and bundled skills controls. The new --safe-mode flag (and CLAUDE_CODE_SAFE_MODE env var) disables all customizations (CLAUDE.md, plugins, skills, hooks, MCP servers) for clean troubleshooting. /cd moves a session to a new working directory without breaking the prompt cache. disableBundledSkills setting hides bundled skills and slash commands from the model.

Bug fixes: enterprise MCP policy enforcement on reconnect, Up/Down arrow navigation in long wrapped lines, macOS UI stall for claude.ai-authenticated users, Windows slash-command scan slowness, Remote Control stuck on reconnecting after session resume, Git Credential Manager popups on Windows startup, stale permission prompts on remote reattachment, background agents ignoring project-level env values, MCPB plugin cache spurious invalidation on Windows, untrusted project settings bypassing OTEL trust confirmation. Reduced CPU during streaming. Restored 5-minute idle timeout on Vertex/Foundry.
How to use claude update. Set CLAUDE_CODE_SAFE_MODE=1 to enforce safe mode in all sessions. Set API_FORCE_IDLE_TIMEOUT=0 to opt out of the Vertex/Foundry idle timeout.

Apps
Claude for Apple Foundation Models framework (announced)
A Swift package that makes Claude available as a server-side language model in Apple's Foundation Models framework. Conforms to the LanguageModel protocol so developers can drive it with LanguageModelSession alongside Apple's on-device model. Supports multi-step reasoning, code generation, web search, code execution, and SwiftUI streaming. Targets iOS 27, iPadOS 27, macOS 27, visionOS 27, watchOS 27. Announced June 9, available June 10.
How to use Add the Swift package. Docs: Apple Foundation Models (Claude API Docs).
