The Lead

Fable 5 is not a Preview. Not behind a waitlist. Anthropic put the first Mythos-class model into general release on June 9: model ID claude-fable-5, 128,000 output tokens, $10 input and $50 output per million tokens. Pro, Max, Team, and seat-based Enterprise plans get it free through June 22. The simultaneous Claude Mythos 5 release stays limited, via Project Glasswing.

API
Claude Managed Agents: Cron Schedules and Credential Vaults (public beta)
Managed Agents now support cron-based scheduling: attach an agent to a schedule on the Claude Platform and it fires automatically, no manual trigger needed. Credential vault integration keeps secrets out of model context entirely, binding keys at the network boundary only on requests to customer-allowlisted domains. No separate scheduler fee; billed through existing Claude Platform usage. Reported by itbrief.news.
How to use Add cron_schedule to your agent definition on the Claude Platform. Store credentials in the platform vault rather than the system prompt. See Claude Platform docs for vault allowlist configuration.
